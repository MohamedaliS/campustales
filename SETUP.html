Quick Start

1. Test Locally (Current Setup)

The site is currently running at:

http://localhost:8000

2. Deploy to GitHub Pages

  1. Commit all files to your repository: 
    git add .
git commit -m "Add Campus Tales with subscription features"
git push origin main
  2. Enable GitHub Pages:
    • Go to repository Settings -> Pages
    • Source: Deploy from branch
    • Branch: main -> / (root)
    • Click Save
  3. Access your site at: https://yourusername.github.io/campustales

Customization

Change Branding

Edit both index.html and admin.html:

<div class="logo-icon">CT</div>
<div class="logo-text">Campus Tales</div>
<div class="author-name">by Mohammed Ali Sharafuddin</div>

Change Admin Password

In browser console:

localStorage.setItem('campusTalesAdminPassword', 'your-new-password');

Modify Colors

Update CSS variables in both HTML files:

:root {
    --accent: #ef4444;        /* Red accent */
    --primary: #000000;       /* Black */
    --background: #ffffff;    /* White */
}

Add New Comics

Edit the comics array in index.html:

const comics = [
    {
        title: "Your Comic Title",
        date: "Month DD, YYYY",
        image: "path/to/image.png",
        caption: "Your caption here"
    },
    // Add more...
];

ðŸ”’ Security Notes

Important: This is a client-side implementation suitable for low-security applications.
  • Admin password stored in localStorage (not production-ready)
  • Subscriber data stored client-side
  • Anyone with browser access can view localStorage

For production with sensitive data, implement:

  • Server-side authentication (Firebase, Supabase)
  • Database for subscriber storage
  • API endpoints with authorization
  • HTTPS encryption

Troubleshooting

Emails not saving

  • Check browser console for errors
  • Verify localStorage is enabled
  • Try in incognito mode

Admin panel not working

  • Verify default password: admin123
  • Clear browser cache
  • Check localStorage in DevTools

Responsive issues

  • Clear browser cache
  • Check viewport meta tag
  • Test in different browsers
